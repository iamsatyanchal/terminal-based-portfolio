# Web Terminal Based Portfolio

A command-line style portfolio website built with React and TypeScript. Navigate through my experience, projects, and skills using familiar terminal commands.

## Commands

- `help` - Show available commands
- `about` - Profile information
- `summary` - Professional summary
- `skills` - Technical skills
- `projects` - Project portfolio
- `experience` - Work history
- `education` - Academic background
- `contact` - Contact details
- `socials` - Social media links
- `ls` - List portfolio sections
- `cat <section>` - View raw data
- `date`, `whoami`, `pwd` - Standard terminal commands
- `clear` - Clear the terminal

## Development

Built with Vite, React, and TypeScript. Data loaded from JSON file for easy updates.
